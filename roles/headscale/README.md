VPN Servers Role
=========

Роль для настройки и управления VPN серверами с использованием Docker.

Requirements
------------

- Ansible 2.9+
- Docker и docker-compose на целевых серверах
- Доступ по SSH к целевым серверам
- тестировалось на ubuntu 20.04 - 22.04 и debian 11 - 12
